# Goodeye Labs

Goodeye turns the business outcomes you care about into verified AI workflows that agents run reliably.

You start from a business outcome, capture the work that moves it as a markdown runbook (a workflow), and pair it with verifiers that score an AI agent's output against a measurable result. The agent runs the workflow, the verifiers judge, and it revises until the output passes. Goodeye reaches your agent on three surfaces: CLI, MCP server, and REST API.
